Property Use Values
opacity Setting this value can change the transparency of an element. A decimal value from 0 to 1. So 50% opactity would be 0.5
visibility Can hide an element, but does not remove it from the page layout flow. Options: visible or hidden
transform Can scale, rotate, and modify an element. Options: scale(2) rotate(45deg). See documentation link below.
transition Specifies how long it takes to transition properties from one state to another. Useful for hover effects or subtle animations. button {transition: all 0.3s ease;} button:hover, button.active {transform: scale(2);}
background Specifies the background image and configuration. Example usage: background: url(../img/bg.jpg) center cover no-repeat fixed;
border Specifies the border around the inside of a box. Example usage: border: 1px solid #ff0000;
color Specifies the color of the text within the element. Value can be English color name red, hex color #ff0000, or rgba(255,0,0,1)
text-align Aligns text within a block element. Options: left, center, right
font Specifies the font family, size, and weight for an element. Normalize will set it to Helvetica/Arial globally. See CSS Properties documentation.
box-shadow Adds a shadow to a block element. Values should be specified in this order: Offset-x Offset-y Blur-radius Spread-radius Color. Example: box-shadow: 3px 10px 5px 2px gray;
text-shadow Adds a shadow to a text element. Values should be specified in this order: Offset-x Offset-y Blur-radius Spread-radius Color. Example: text-shadow: 3px 10px 5px 2px gray;
