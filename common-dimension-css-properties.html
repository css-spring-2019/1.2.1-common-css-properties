Property Use Values
padding Adds space around content inside a box. Dimensions in px, %, or vw. TOP RIGHT BOTTOM LEFT ordering: padding: 10px 20px 30px 40px;
margin Adds space around content outside a box. Same as padding.
width Width of box. Works only on block elements, not inline. Dimension in px, %, or vw generally.
min/max-width Minimum or maximum allowable width. -
height Height of box. Works only on block elements. Dimensions cannot be in % generally.
min/max-height Minimum or maximum allowable height. -
transform Can scale, rotate, and modify an element. Options: scale(2) rotate(45deg). See documentation link below.
